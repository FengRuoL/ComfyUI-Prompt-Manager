## 1. 仓库简介

这里是 ComfyUI-Prompt-Manager 插件的官方**社区交互中心**。

为了突破容量限制、提供极速的下载体验，本项目的**实体海量图库数据已全面迁移并托管至 [Hugging Face Dataset] (https://huggingface.co/datasets/FRuoL/ComfyUI-Prompt-CloudDB/discussions) **。
所有安装了该插件的用户，将通过插件底层的直连通道，在 ComfyUI 界面中无感、极速地浏览由社区共同维护的高质量 Prompt 预设、画师风格库。

本 GitHub 仓库目前作为**前端社区版块**，专门用于接收用户的 Bug 反馈、功能建议以及 **Prompt 词库的开源投稿**。

## 2. 如何参与内容贡献 (在此提交 Issues)

我们非常欢迎并鼓励社区用户将自己整理的优质 Prompt 或画师风格提交至此，共同丰富公共图库！
为降低技术门槛，我们坚持采用 **GitHub Issue 投稿制**，您无需掌握 Git 代码提交流程即可完成贡献。

根据您提交的数据量，请选择以下对应的投稿方式：

### 方案 A：提交单条或少量预设
如果您在日常使用中发现了 1-3 个非常优秀的画师风格或提示词预设，请按以下步骤操作：
1. 点击本页面顶部的 **Issues** 标签页。
2. 点击绿色的 **New issue** 按钮。
3. 选择 **"提交新的画师串/Prompt预设"** 模板。
4. 按照模板内的提示，填入画师名称、Prompt 内容、建议的分类，并**必须**拖入 1-3 张 ComfyUI 生成的预览图。
5. 提交 Issue，等待管理员审核录入。

### 方案 B：批量提交整个分类或图库包 (强烈推荐)
如果您在本地整理了几十个画师、建立了一整套完整的分类体系，并希望一次性贡献出来，请利用插件自带的打包功能：
1. 在您本地的 ComfyUI 中，使用 Prompt 浏览器整理好分类、文本并上传好所有预览图。
2. 在插件侧边栏点击 **"导出配置"**。
3. 选择需要导出的范围，并务必勾选 **"包含图片数据"** (这很重要，否则没有预览图)。
4. 插件将生成一个包含完整目录结构和图片数据的 JSON 文件。
5. 将该文件打包为 ZIP 格式（或将后缀改为 TXT 以便 GitHub 支持上传）。
6. 回到本页面的 Issues 页面，选择 **"批量提交/整体图库打包"** 模板，将文件拖入并在说明中简要描述您的图库内容，提交即可。

*注：管理员收到您的文件后，会在本地审查通过后，统一同步推送至 Hugging Face 云端主库，实现全网瞬间更新。*

## 3. 上传与内容规范要求

为保证公共图库的质量与合规性，请在提交前务必确认您的内容符合以下标准。违反以下规范的提交将被直接关闭并拒绝收录：

* **R18 与 NSFW 尺度限制**：**严格禁止**提交任何包含生殖器官暴露、真实性行为描写、极端暴力血腥、儿童色情或违反相关法律法规的内容与图片。对于因模型原因产生的轻度擦边内容（如普通泳装、内衣等），管理员将视具体尺度进行自由裁量审核。
* **图片质量要求**：预览图需具备代表性，能够准确反映该 Prompt 或画师风格的特征。请勿上传与 Prompt 描述无关的占位图。
* **参数纯净度**：提交的 Prompt 请尽量保持纯净，移除如视角控制、特定人物特征等强关联私有设定的词汇，仅保留与“画风、光影、笔触、核心概念”相关的通用提示词。
* **版权与争议**：请勿恶意冒用他人已发布的整理包并声明为原创。如引用他人公开的风格测试表，请在 Issue 说明中附带原作者出处或致谢链接。
